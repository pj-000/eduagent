# 工作流设计报告

> 生成时间：2026-03-05 20:19:45
> 数据来源：custom_goal

## 工作流目标

实现三角函数概念理解、技能训练与学习成效闭环评估的教学工作流

## 子任务列表

### 1. diagnose_prior_knowledge

**描述**：评估学生关于角、单位圆、直角三角形等前置知识的掌握情况

**输入**：
- `student_grade_level` (string): 学生年级（如'高一'）
- `curriculum_standards` (string): 对应学段课程标准条目

**输出**：
- `knowledge_gaps` (string): 识别出的具体前置知识薄弱点列表

**Agent Prompt**：

```
你是教育诊断专家，专精于数学前概念评估。请结合{student_grade_level}学生的认知发展特点和{curriculum_standards}中的要求，分析学生在学习三角函数前最可能存在的知识断层（如弧度制理解、单位圆坐标意义、相似三角形比例关系等）。请仅输出结构化知识薄弱点列表，每项用分号分隔。
```

---

### 2. design_conceptual_sequence

**描述**：基于诊断结果设计符合认知逻辑的三角函数教学序列

**输入**：
- `knowledge_gaps` (string): 前置知识薄弱点列表

**输出**：
- `teaching_sequence` (string): 含情境导入、具象操作、符号抽象、应用迁移四阶段的教学流程描述

**Agent Prompt**：

```
你是数学教学设计专家，遵循CPA（具体-图像-抽象）教学原则。请根据以下前置知识薄弱点：{knowledge_gaps}，设计一个45分钟三角函数入门课的教学序列，必须包含：1）生活化情境导入；2）单位圆动态操作活动；3）正弦/余弦比值定义的符号化提炼；4）简单实际问题迁移。请用清晰阶段标题+一句话说明的形式输出。
```

---

### 3. generate_scaffolded_practice

**描述**：生成匹配教学序列的渐进式练习题集（含可视化提示与错误诊断反馈）

**输入**：
- `teaching_sequence` (string): 四阶段教学流程描述

**输出**：
- `practice_set` (string): 含三组题（基础辨识、变式计算、开放建模）及对应诊断反馈提示

**Agent Prompt**：

```
你是资深数学学科命题专家，擅长设计支架式练习。请依据以下教学流程：{teaching_sequence}，生成三组递进练习：第一组聚焦单位圆坐标的直观辨识（配图提示），第二组训练角度与比值的双向转换（含常见错误选项），第三组要求用三角函数建模真实场景（如摩天轮高度变化）。每组题后附1条针对性诊断反馈提示。请严格按‘【基础】...【反馈】...【变式】...【反馈】...【建模】...【反馈】’格式输出。
```

---

### 4. administer_formative_assessment

**描述**：实施形成性测验并实时分析作答模式与典型迷思

**输入**：
- `practice_set` (string): 渐进式练习题集

**输出**：
- `misconception_report` (string): 识别出的学生高频迷思类型及证据示例（如‘正弦值随角度增大而单调递增’）

**Agent Prompt**：

```
你是教育测评数据分析专家。请模拟对30名学生施测以下练习题集：{practice_set}，基于常见三角函数学习迷思库，分析作答数据中出现频率最高的3类概念性错误，每类需注明：错误表述、出现频次估计、典型作答示例（虚构但合理）。请用‘错误类型：...；证据：...’格式分条列出。
```

---

### 5. adapt_instructional_response

**描述**：针对迷思报告生成个性化补救教学策略与微资源推荐

**输入**：
- `misconception_report` (string): 高频迷思类型及证据示例

**输出**：
- `intervention_plan` (string): 含矫正活动设计、类比解释话术、交互式微课链接的个性化干预方案

**Agent Prompt**：

```
你是精准教学干预专家。请针对以下迷思报告：{misconception_report}，为每位迷思类型设计：1）一个5分钟课堂矫正活动（含教具或板书草图说明）；2）一句直击本质的类比解释话术（如‘正弦值不是角度的影子，而是单位圆上点的y坐标’）；3）一个匹配该迷思的优质交互式微课资源名称及平台（如‘Desmos单位圆动态追踪器’）。请按迷思类型分块输出，每块含三项内容。
```
