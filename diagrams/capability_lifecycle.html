② 两类能力 Artifact 对比

⚡ ExecutableTool（可执行工具）

  • Python 函数，JSON 序列化 I/O
  • 在 Sandbox 中受限执行
  • 白名单模块：math / random / json / re / collections 等
  • 禁止：os / subprocess / requests / open / eval
  • smoke_test 无参调用验证
  • 示例：generate_math_problems, english_fill_blank

💡 PromptSkill（提示词技能）

  • 上下文注入策略，像 Claude Code skill
  • trigger_guidance：关键词匹配触发
  • prompt_fragment：注入 Agent system prompt
  • allowed_tools：限定可调用工具
  • 确定性选择（关键词匹配打分）
  • 示例：text_simplifier（文本简化策略）
